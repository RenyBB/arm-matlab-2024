What is MATLAB?

  • Matlab is a high-level programming language that is specialised for dealing with matrices. The name Matlab comes from Matrix Laboratory. When referring to programming languages, high-level means that there is a high level of abstraction from the instruction set the computer is using. High-level languages look more like natural human language.

  • While not as flexible as some other languages (e.g., Python), Matlab makes it quick and easy to write advanced programs for the kind of tasks we want to do in Psychology research. For example, signal processing (EEG research), image processing (vision research), and mathematically modelling (psychophysics and signal detection).

  • This is because many popular toolboxes have been implemented in Matlab

    • FieldTrip: EEG, MEG

    • SPM: EEG, MEG, fMRI, PET, DCM

    • EEGLab: EEG

    • PsychToolbox: For running experiments

Most of these packages are for analysing data matrices
